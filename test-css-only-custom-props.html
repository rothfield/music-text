How it works (CSS only!):
/* Define the tuplet styling */
.tuplet::before {
    content: var(--tuplet-number);  /* Gets "3", "5", "7", etc. */
    width: var(--tuplet-width);     /* Gets 3ch, 5ch, 7ch, etc. */
}

/* Set the properties in CSS */
.triplet {
    --tuplet-number: "3";
    --tuplet-width: 3ch;
}

/* Or set them inline */
<span class="tuplet" style="--tuplet-number: '23'; --tuplet-width: 23ch;">
